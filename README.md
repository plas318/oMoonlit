# oMoonlit
oMoonlit is a private blog- website project created using django (drf) rest api and react

## Frameworks
- Django, Djangorestframework
- DRFspectacular
- React
- Hostinger - Domain
- Amazon AWS - Server / Routing
  
## Getting Started

To run a local backend server :
`python manage.py runserver`
